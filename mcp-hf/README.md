# 🔌 Model Context Protocol (MCP) — Hugging Face Course

Notes and examples from the **[Hugging Face MCP Course](https://huggingface.co/learn/mcp-course)**
(built in partnership with **Anthropic**). MCP is the open standard that connects
AI models to external tools and data — *"the USB-C for AI"*.

## 🧠 What I'm learning
- The **M×N → M+N** integration problem MCP solves.
- MCP architecture: **Host · Client · Server**.
- Server capabilities: **Tools · Resources · Prompts · Sampling**.
- Building and connecting MCP servers/clients.

## 📚 Contents
- [`APUNTES.md`](APUNTES.md) — chapter-by-chapter study notes (ES).
- [`code/server.py`](code/server.py) — a small **MCP server** (FastMCP) exposing text-utility
  tools + a resource + a prompt. Test with `mcp dev server.py`.
